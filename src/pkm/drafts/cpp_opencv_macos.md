
compile c++ opencv on macos 

```cpp
g++ -g -std=c++11 -I/usr/local/include/opencv4 -L/usr/local/lib -lopencv_core -lopencv_highgui -lopencv_imgproc -lopencv_imgcodecs main.cpp -o main
```

export DYLD_LIBRARY_PATH=/usr/local/lib
./main
